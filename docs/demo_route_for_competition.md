# 竞赛演示路线

本文档为光学薄膜教学平台竞赛演示提供路线规划。

## 5 分钟演示路线

### 目标
快速展示平台核心功能，让评委了解平台能力。

### 路线

1. **开场** (30s)
   - 平台名称和定位
   - "基于 Python 与传输矩阵法（TMM）的薄膜光学计算实验与教学平台，不依赖外部商业软件运行时"

2. **核心功能演示** (2min)
   - 运行 `run_teaching_demo.py --case single_ar`
   - 展示 R/T/A 光谱图及物理一致性自检（能量守恒 $R+T+A \equiv 1.0$）
   - 展示平面膜系在一维空间中的电场局域分布图

3. **工程应用案例** (1min)
   - 展示太阳能电池 AR 案例、相机镜头 AR 对照
   - 强调"从基础教学现象物理演示到光电工程实际设计指标分析"的覆盖

4. **交互性展示** (1min)
   - 展示 Plotly 交互图表（支持浮动查看物理量和无级缩放）
   - 展示在网页端/数据报包中查看参数调节（厚度、角度）的物理效应

5. **技术亮点** (30s)
   - 270 个 pytest 单元测试保证计算物理正确性与数值稳定性
   - 传输矩阵法（TMM）向量化并行级联重构，相较传统循环大幅提升计算效率
   - 周期性结构的一维极化等效介质理论 (EMT) 近似模块

### 关键台词
- "本平台完全基于底层物理公式进行编码实现，脱离了对商业软件运行时的依赖"
- "所有的电磁场边界递推均通过了能量守恒和退化极限测试验证，确保理论闭环"
- "实现了从基础物理原理演示到复杂光电器件工程设计的复验与演示"

## 15 分钟答辩路线

### 目标
深入展示技术细节、底层物理体系和创新点。

### 路线

1. **问题背景与选题意义** (2min)
   - 光学薄膜在光电制造与微纳器件中的重要地位
   - 高校波动光学教学中公式繁多、电磁场空间分布抽象的痛点
   - 开源交互式轻量级工具对低成本实验教学的巨大补充价值

2. **平台底层物理体系** (2min)
   - 麦克斯韦方程与一维切向连续电磁边界条件
   - 菲涅尔公式级联与 TMM 特征矩阵推导
   - 真实材料色散（Cubic Spline 插值库）与消光系数吸收模型
   - 亚波长周期结构一维极化等效介质理论 (EMT)

3. **核心算法设计** (3min)
   - TMM 特征矩阵的 NumPy 向量化重构，消除多重循环
   - 真实材料双级缓存与读盘机制
   - 物理一致性测试套件的自动化验证

4. **工程应用与现象复验** (3min)
   - 5 个 TMM-only 工程应用案例（太阳能、通信滤波、激光镜、相机镜头、节能窗）
   - 前沿拓展研究案例（被动辐射制冷 PDRC 的 ASTM G173 太阳光谱加权、Tamm 局域态相位差筛选）

5. **可靠性验证与复现机制** (2min)
   - pytest 单元测试及断言覆盖（能量守恒、极值退化、偏振分裂）
   - 一键自检程序与数据归档（docs/evidence/）

6. **技术边界与创新点** (2min)
   - 厘清平台物理边界（一维周期结构部分为零阶 EMT 近似，并不对应严格耦合波分析算法的特征值求解，也不替代商业软件的内部全波求解）
   - 创新点汇报（开源轻量、理论设计一体化、色散与案例结合、自动化测试、教育模块等）

7. **未来展望** (1min)
   - 计划后续拓展周期结构物理分析能力
   - 进一步合理扩充微纳光子器件工程设计案例

## 亚波长一维周期结构 EMT 极化分析路线

### 当前状态
- 一维周期性亚波长结构有效介质理论 (EMT) 极化计算
- 支持 TE/TM 各向异性偏振对照
- 19 个关于该模块的专项单元测试

### 后续扩展
1. 完整一维周期衍射求解方法开发计划
2. 二维周期结构格点支持
3. COMSOL 联合扫描光谱数据的深入特征提取

### 演示代码
```python
from guided_grating import rcwa_1d, GratingLayer, rcwa_convergence_test
import numpy as np

# 基本计算
g = GratingLayer(period_nm=980, thickness_nm=200, n_low=1.45, n_high=3.4, fill_factor=0.55)
result = rcwa_1d(np.linspace(1400, 1700, 100), g, pol="TE")

# 极化计算收敛性测试
conv = rcwa_convergence_test(g, wavelength_nm=1550.0)
print(f"EMT 计算收敛状态: {conv['converged']}")
```

## 物理与技术边界声明

### 平台定位
- **教学展示平台**：完全基于 Python 底层物理规律构建。
- **不替代商业求解器**：平台本身不执行空间有限元等数值离散求解。COMSOL 仅用于前期复杂器件参数探索后的数据比对。

### 答辩/演示话术
- "本平台采用独立 Python 算法代码实现，降低了波动光学计算和物理设计的软硬件门槛。"
- "本平台平面部分以传输矩阵法 (TMM) 为计算核心；亚波长周期结构部分采用零阶有效介质理论 (EMT) 进行等效。我们严格厘清了平台的物理计算边界，不夸大宣传。"
- "在遇到复杂的三维电磁散射或光栅波导共振时，平台设计了 CSV 数据管线，直接读取并特征提取 COMSOL 数据，以互补的形式实现学术复验。"
