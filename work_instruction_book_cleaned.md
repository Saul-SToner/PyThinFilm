### 一、项目介绍

#### 1.1 创作背景

##### 1.1.1 薄膜光学及其重要性
薄膜光学是波动光学的重要分支，研究电磁波通过单层或多层介质薄膜时的干涉、衍射和吸收行为。随着精密制造与微纳光子学的快速发展，光学薄膜已广泛应用于半导体芯片刻蚀、太阳能电池涂层、光通信器件、激光高反镜以及建筑节能窗户等工程领域。理解并设计复杂的薄膜结构，对于现代光电信息技术的发展具有支撑作用。

##### 1.1.2 高校教学与实验中的现实困难
在高校《波动光学》及相关物理课程中，多层膜干涉的解析计算公式多、概念抽象。例如，随着薄膜层数增加，菲涅尔反射系数在多个界面的相长或相消叠加过程空间电场分布变得难以直观推导。学生通常仅能接触到少层数的理想薄膜物理结论，而对于实际色散薄膜及微纳光栅结构的电磁特性缺乏直观感知。此外，测量多层膜光谱所需的光学仪器（如椭偏仪、高真空反射式镀膜机）造价较高，由于实验室安全和资金限制，难以大规模作为本科生自主实验项目开展，导致理论教学与工程实践之间存在脱节。

##### 1.1.3 Python 开源仿真平台的解决思路
为了解决上述物理图像抽象与实践条件限制的矛盾，开发一款轻量化、开源、可视化且计算逻辑可查的薄膜仿真工具是一条有效途径。Python 作为通用的开源编程语言，具备科学计算与图形交互生态。通过自主编写计算内核，可以实现无需依赖大型商业软件的特征矩阵计算平台，为高校教学和工程设计提供轻量化补充方案。

#### 1.2 目标定位
本项目旨在构建一整套基于开源 Python 环境开发的**光学薄膜仿真、设计与可视化教学系统**（版本定位为 `v0.2.1 竞赛展示功能冻结版`）。
* 本平台利用特征矩阵算法，自主实现了平面多层薄膜系统的反射率、透射率、吸收率及腔内电场空间分布的求解。
* 针对周期性亚波长结构，平台引入了有效介质理论（EMT）近似分析模块，用于演示各向异性极化特性。
* 系统内部集成了标准教学体系下的多种典型膜系、多个工程应用设计案例，以及面向被动辐射冷却与 Tamm 界面态分析的拓展研究示例，并通过交互式 Plotly 图表展示物理效果。

#### 1.3 选题意义
本项目通过编写计算代码，将物理规律转化为可视化交互。该项目具有以下选题意义：
1. **科教融合探索**：通过将 Tamm 界面态、PDRC 被动制冷薄膜等拓展研究示例合理简化，并集成入可视化仿真模块，有助于拓宽基础物理教学的视野。
2. **降低学习与设计门槛**：基于 Python 代码计算，学生可通过运行轻量级 Python 脚本完成薄膜光谱扫描与参数分析，降低了实验的软硬件门槛。
3. **践行开源与技术可信**：平台全部算法开源，算法透明。在文档和说明中明确厘清了 TMM 求解与亚波长光栅 EMT 近似分析模块的物理边界，并通过测试套件对关键环节进行验证，提高计算流程的可复现性与可靠性。

### 二、开发平台与理论基础

#### 2.1 薄膜光学基本原理

##### 2.1.1 光在介质界面的反射与透射
一束平面电磁波以角度 $\theta_i$ 从介质 1（折射率 $n_1$）入射到介质 2（折射率 $n_2$）的平面分界界面上时，会在分界面发生反射与透射。电磁场在分界面处的切向分量必须满足连续边界条件，即电场和磁场的切向分量相等。

##### 2.1.2 菲涅尔公式
在分界面上，根据电场振动方向与入射面（入射光线与界面法线构成的平面）的关系，平面波可分为两种偏振状态：
* **TE 偏振**（S 偏振）：电场方向垂直于入射面。
* **TM 偏振**（P 偏振）：电场方向平行于入射面。

根据电磁边界条件，这两种偏振对应的 Fresnel 振幅反射系数 $r$ 和振幅透射系数 $t$ 分别为：
$$r_s = \frac{n_1 \cos\theta_i - n_2 \cos\theta_t}{n_1 \cos\theta_i + n_2 \cos\theta_t}, \quad t_s = \frac{2n_1 \cos\theta_i}{n_1 \cos\theta_i + n_2 \cos\theta_t}$$
$$r_p = \frac{n_2 \cos\theta_i - n_1 \cos\theta_t}{n_2 \cos\theta_i + n_1 \cos\theta_t}, \quad t_p = \frac{2n_1 \cos\theta_i}{n_2 \cos\theta_i + n_1 \cos\theta_t}$$
其中 $\theta_t$ 为折射角，由折射定律 $n_1 \sin\theta_i = n_2 \sin\theta_t$ 确定。

##### 2.1.3 多光束干涉与薄膜相位差
若介质为平面薄膜，当厚度与波长可比拟时，电磁波在薄膜的上下两个分界面处会发生无限次往复反射与透射，反射波在膜层上表面发生干涉。相邻两次出射的干涉光波的光程差决定了总反射光谱。光波在第 $j$ 层介质中单次往返引起的光学相位延迟 $\delta_j$ 表示为：
$$\delta_j = \frac{2\pi}{\lambda} \tilde{n}_j d_j \cos\theta_j$$
其中 $\tilde{n}_j(\lambda) = n_j(\lambda) - i\kappa_j(\lambda)$ 为该层介质的复折射率，$d_j$ 为薄膜物理厚度，$\theta_j$ 为膜层内的折射角。

#### 2.2 传输矩阵法 TMM

##### 2.2.1 单层膜特征矩阵
传输矩阵法通过一维边界条件级联，求解多层平面薄膜的场关系。对于第 $j$ 层平面介质薄膜，其左边界 $j-1$ 处的切向场（电场 $E_{j-1}$，磁场 $H_{j-1}$）与右边界 $j$ 处的切向场（$E_j$，$H_j$）通过特征矩阵 $M_j$ 关联：
$$\begin{pmatrix} E_{j-1} \\ H_{j-1} \end{pmatrix} = \begin{pmatrix} \cos\delta_j & \frac{i}{\eta_j}\sin\delta_j \\ i\eta_j\sin\delta_j & \cos\delta_j \end{pmatrix} \begin{pmatrix} E_j \\ H_j \end{pmatrix} = M_j \begin{pmatrix} E_j \\ H_j \end{pmatrix}$$
其中 $\delta_j$ 为前述相位延迟，$\eta_j$ 为第 $j$ 层介质的斜入射光学导纳。
对于 TE 偏振：
$$\eta_{j,TE} = \tilde{n}_j \cos\theta_j$$
对于 TM 偏振：
$$\eta_{j,TM} = \frac{\tilde{n}_j}{\cos\theta_j}$$

##### 2.2.2 多层膜总矩阵
当系统由 $N$ 层平面介质堆叠组成时，通过连续性边界条件将各层界面的特征矩阵依次级联，即可得到整个系统的总传输特征矩阵 $M$：
$$M = \prod_{j=1}^{N} M_j = M_1 M_2 \cdots M_N = \begin{pmatrix} m_{11} & m_{12} \\ m_{21} & m_{22} \end{pmatrix}$$

##### 2.2.3 反射率、透射率、吸收率计算
设入射介质的光学导纳为 $\eta_0$，基底介质的光学导纳为 $\eta_s$。振幅反射系数 $r$ 和振幅透射系数 $t$ 可由下式计算：
$$r = \frac{\eta_0 (m_{11} + \eta_s m_{12}) - (m_{21} + \eta_s m_{22})}{\eta_0 (m_{11} + \eta_s m_{12}) + (m_{21} + \eta_s m_{22})}$$
$$t = \frac{2\eta_0}{\eta_0 (m_{11} + \eta_s m_{12}) + (m_{21} + \eta_s m_{22})}$$
由此可计算出反射率 $R$、透射率 $T$ 和吸收率 $A$：
$$R = |r|^2$$
$$T = \frac{\text{Re}(\eta_s)}{\text{Re}(\eta_0)} |t|^2$$
$$A = 1 - R - T$$

##### 2.2.4 TE/TM 偏振差异
斜入射时（入射角 $\theta_0 > 0$），由于 TE 和 TM 偏振对应的光学导纳 $\eta_j$ 的物理定义不同（TE 正比于 $\cos\theta_j$，TM 反比于 $\cos\theta_j$），这导致了光谱透射和反射在两种极化状态下表现出各向异性，例如在特定入射角（如布儒斯特角）下 TM 偏振反射率会发生急剧衰减甚至归零。

##### 2.2.5 腔内电场分布计算
在计算出各界面的总反射系数和透射系数后，可以通过一维反向递推，计算出多层膜内部每一层不同深度 $z$ 处的正向波和反向反射波的相干叠加场强。内部第 $j$ 层深度为 $z'$ 处的电场驻波分布可描述为正向与反向传输电磁分量的干涉：
$$E_j(z') = E_{j}^+ e^{-i \tilde{k}_{jz} z'} + E_{j}^- e^{i \tilde{k}_{jz} z'}$$
其中 $\tilde{k}_{jz} = \frac{2\pi}{\lambda}\tilde{n}_j\cos\theta_j$。通过这一公式可以直观展示一维光学谐振腔内的场分布特性。

#### 2.3 真实材料色散

##### 2.3.1 nk 数据的意义
实际的光学材料折射率在整个光谱段并非理想常数，而是伴随波长发生色散变化。此外，金属、半导体等材料在特定波长下存在固有的带间跃迁吸收，其复折射率中的虚部 $\kappa(\lambda)$ （消光系数）不可忽略。引入真实折射率和消光系数数据，是薄膜性能偏离理想状态分析的前提。

##### 2.3.2 材料插值与缓存
本平台真实材料库集成自 RefractiveIndex.INFO 的实测物理常数数据，数据以 `.csv` 格式分类存放。在运行仿真时，算法使用单次读盘机制和内存字典缓存以避免重复读取，并利用三阶样条插值算法将材料在离散采样点的 $n$ 和 $\kappa$ 值，平滑插值到仿真波长网格上，以消除常数近似带来的模拟谱偏离。

#### 2.4 亚波长光栅 EMT 近似

##### 2.4.1 有效介质理论
对于一维周期结构光栅，当其周期 $P$ 远小于入射电磁波长 $\lambda$ （处于亚波长静态区，即 $P \ll \lambda$）时，光栅层可等效为一个均质各向异性薄膜层。这就是有效介质理论（Effective Medium Theory, EMT）的物理边界。

##### 2.4.2 TE/TM 等效折射率
在零阶有效介质理论近似下，对于平行（TE 偏振）和垂直（TM 偏振）于条纹方向的电矢量入射，光栅层可等效为以下均质折射率层：
$$n_{eff,TE}^2 = f n_{high}^2 + (1 - f) n_{low}^2$$
$$\frac{1}{n_{eff,TM}^2} = \frac{f}{n_{high}^2} + \frac{1 - f}{n_{low}^2}$$
其中 $f$ 为高折射率材料所占的体积占空比（Fill Factor），$n_{high}$ 和 $n_{low}$ 分别为组成光栅的高、低折射率材料折射率。本平台基于此等效值，使用传输矩阵法计算单层各向异性薄膜，以演示亚波长格栅的偏振干涉特性。

##### 2.4.3 适用边界与失效条件
* **适用条件**：必须严格满足 $P/\lambda \to 0$。通常在 $P/\lambda \le 0.1$ 时，零阶 EMT 近似与严格物理仿真有较好的一致性。
* **失效条件**：当周期 $P$ 增大，与工作波长 $\lambda$ 可比拟（例如 $P/\lambda \ge 0.5$）时，高阶衍射级次被激发并在结构中传播，形成明显的波导共振或衍射分支。此时零阶 EMT 物理模型不再适用，需要采用更高阶的周期结构电磁求解方法。

##### 2.4.4 与严格 RCWA 的区别
严格耦合波分析（RCWA）通过将麦克斯韦方程在傅里叶空间展开，解算包含无数个衍射级次相互作用的矩阵特征值系统。而当前的 EMT 模块仅提取零阶有效双折射各向异性物性进行一维传输求解。
guided_grating/rcwa.py 文件名沿用了早期开发阶段的模块命名；当前 v0.2.1 版本中，该模块的物理实现边界为 EMT 近似分析，不对应严格 RCWA 算法实现。

#### 2.5 外部仿真数据导入

##### 2.5.1 COMSOL CSV 数据读取
为了展示偏振共振光栅等超出 EMT 近似适用范围的周期结构特性，平台设计了外部 CSV 数据解析读取管线。COMSOL 仅作为外部高精度仿真数据的来源，本平台不依赖 COMSOL 运行时或授权，可以在独立 Python 环境中完成光谱数据分析。

##### 2.5.2 光谱特征提取
通过 Pandas 解析包含复杂表头的 CSV 数据列，算法在导入数据后自动对反射/透射曲线进行滤波和二阶导数寻找极值点，实现特征光谱信息的自动解算。

##### 2.5.3 FWHM、Q 值、MAE 等指标
特征提取模块可自动计算共振峰在特定阈值下的半高全宽（FWHM），估算谐振腔的品质因子 $Q$ 值（定义为 $Q = \lambda_0 / FWHM$），以及计算理论曲线与测量数据之间的平均绝对误差（MAE）和均方根误差（RMSE），以此作为器件设计的参数筛选指标。

```text
└── smoke_test.py                # 快速功能自检脚本
```

#### 3.3 核心模块说明
* **thinfilm 核心计算模块**：包含 `thinfilm/education.py` 和 `thinfilm/api.py`。核心计算内核基于一维多层薄膜特征矩阵算法，将波长轴和角度轴 NumPy 向量化广播重构，完成高效的 R/T/A 及腔内一维电场强度求解。
* **materials 材料模块**：位于 `thinfilm/materials.py`。管理 `data/real_nk/` 内的数据，使用全局 manifest 单次读盘机制，引入三阶样条平滑插值以及 LRU 缓存以优化计算效率。
* **education 教学模块**：位于 `thinfilm/education.py`。内置了教学体系下的单层、多层膜经典参数配置，并提供了参数说明信息和公式库查询字典。
* **guided_grating 拓展模块**：包含 `guided_grating/rcwa.py` 和 `guided_grating/comsol_io.py`。该模块提供基于极化等效介质理论（EMT）的一维周期结构反射率计算。该文件名沿用了早期开发阶段的模块命名，但当前版本中其物理实现边界为 EMT 近似分析。同时提供用于导入和分析外部 COMSOL 仿真 CSV 的解析器。
* **examples/applications 工程案例模块**：位于 `examples/applications/`。包含 5 个 TMM-only 工程应用案例，主要由 Python 底层 TMM 计算模块驱动，展示了薄膜光学在太阳能减反、激光高反、通信滤波等实际场景的应用。
* **tests 自动化测试模块**：位于 `tests/` 目录。包含由 11 个大类测试文件构成的 270 个单元用例。对 TMM 极化各向同性一致性、特征矩阵单层等效厚度退化、能量守恒性、色散插值及 IO 读取异常进行了较为系统的测试。

#### 3.4 数据流与计算流程
【图 3-2：平台仿真计算与数据导出数据流向图】
平台典型仿真流向如下：
1. 用户输入几何厚度、材料组配与波长范围；
2. API 调用 `thinfilm/materials.py` 加载并插值复折射率 $\tilde{n}_j(\lambda) = n_j(\lambda) - i\kappa_j(\lambda)$；
3. 传输矩阵求解器计算特征矩阵并返回 S 和 P 偏振下的 $R(\lambda), T(\lambda), A(\lambda)$ 曲线；
4. 腔内场解析模块正反向递推求解电场驻波分布；
5. 可视化模块将电场与光谱数据生成 Plotly 图表，并输出 CSV/JSON 指标报包到主目录 `~/thinfilm_outputs` 下。

#### 4.1 教学基础仿真模块展示

##### 4.1.1 单层增透膜
* **案例背景**：为提高镜片透射率而广泛使用的单层干涉薄膜。
* **物理原理**：利用薄膜上下两个界面反射光的相位差实现相消干涉。当薄膜的光学厚度 $n d = \lambda_0 / 4$ 且折射率满足 $n = \sqrt{n_0 n_s}$ 时，在中心波长 $\lambda_0$ 处，上下表面的 Fresnel 振幅反射系数相位相差 $\pi$，产生反射相消。
* **结构与参数**：入射介质折射率 $n_0 = 1.0$，基底折射率 $n_s = 1.52$（玻璃），薄膜材料为 MgF2（理想折射率 $n = 1.38$，厚度为 $99.6$ nm），设计波长 $\lambda_0 = 550$ nm。
* **平台计算流程**：调用 `simulate_report_design("single_ar")` 计算 $300\sim800$ nm 反射谱。
* **输出结果说明**：在 $550$ nm 处反射率出现极小值，相比于裸玻璃界面的 $4.26\%$ 反射率，减反射薄膜反射率显著下降。
* **教学或工程意义**：验证了 1/4 波长条件和振幅匹配条件对极小值干涉行为的影响。
* **图示预留位置**：【图 4-1：单层增透膜光谱反射率计算图】

##### 4.1.2 多层介质高反膜
* **案例背景**：精密干涉仪与固体激光器谐振腔所需的高反射镜。
* **物理原理**：采用高低折射率交替的 1/4 波长厚度膜系，各分界面的反射光回到最外表面时，由于相位延迟均为 $\pi$ 的整数倍，呈现同相位，产生相长干涉，反射率随周期数增加呈现上升趋势。
* **结构与参数**：基底折射率 $n_s = 1.52$，交替膜层为高折射率 $n_H = 2.35$、低折射率 $n_L = 1.38$，每层光学厚度均为 $\lambda_0 / 4$。
* **平台计算流程**：运行 `simulate_report_design("high_reflector")`。
* **输出结果说明**：在设计中心波长 $\lambda_0 = 500$ nm 附近，反射光谱形成平坦的高反射带，随着交替周期数增加，高反带中心反射率最终趋近于 99.9%。
* **教学或工程意义**：展示了相长干涉累加物理图像，预测了布拉格反射镜反射停带（Bandgap）的物理边界。
* **图示预留位置**：【图 4-2：多层介质高反射膜的停带宽度与反射率谱图】

##### 4.1.3 法布里-珀罗腔
* **案例背景**：窄带光谱过滤与微腔共振的核心器件。
* **物理原理**：由两组布拉格反射膜堆构成对称分布的端反射镜，中间腔层厚度为 $\lambda_0 / 2$ 的共振介质空腔层。在多光束干涉往返传输中，满足往返相对相位差 $\Delta \phi = 2m\pi$ 的驻波分量发生相长干涉，产生透射。
* **结构与参数**：两端高反镜各为 5 对 H/L 1/4波长膜堆，中间腔层厚度为 $\lambda_0 / 2$ 的低折射率介质。
* **平台计算流程**：运行 `simulate_report_design("fp_filter")` 计算光谱。
* **输出结果说明**：在反射停带的中心出现了一条半高宽窄至几纳米的透射共振峰，对应的透射率峰值接近 1.0。
* **教学或工程意义**：展示了光学微腔的透射特征，揭示了自由光谱范围（FSR）和线宽的决定因素。
* **图示预留位置**：【图 4-3：F-P 腔干涉共振峰分布曲线】

##### 4.1.4 TE/TM 偏振对比
* **案例背景**：演示非正入射条件下的偏振各向异性。
* **物理原理**：斜入射时，电磁场的切向边界条件使得 TE 和 TM 极化的斜入射光学导纳不同，因此反射谱的带宽与布儒斯特角等特征呈现出偏振敏感性。
* **结构与参数**：单层 MgF2 薄膜在玻璃上，入射角设为 45 度。
* **平台计算流程**：调用 TMM 计算 $0\sim90$ 度入射角下的偏振反射谱。
* **输出结果说明**：TE 偏振反射率随角度单调增加，而 TM 偏振反射率在偏振极小角度发生跌落。
* **教学或工程意义**：直观展示了偏振差异及布儒斯特角的物理特征，加深对电磁边界条件的理解。
* **图示预留位置**：【图 4-4：斜入射下 TE/TM 偏振光谱偏振各向异性对照图】

##### 4.1.5 腔内电场分布
* **案例背景**：理解光学薄膜内的场局域与能量分布。
* **物理原理**：反射相干干涉在空间上表现为前向电磁波与反向电磁波构成的电场驻波分布，腔体内电场场强的空间局域峰值位置与谐振腔模式相对应。

### 五、作品分析

#### 5.1 计算结果分析
通过与外部参考数据及文献解析解的对比，本平台在标量及色散 TMM 解析计算上的计算误差处于数值噪声级。在微纳光栅中，亚波长极限条件（$P \ll \lambda$）下 EMT 的零阶反射率趋势表现出较好的预测性。

#### 5.2 教学价值分析
平台代码开源且物理原理透明，避免了商业软件的“技术黑箱”。学生在调节厚度、折射率后，能够迅速在图形中看到干涉移动，有助于加深对波动光学干涉理论的具象化认知。

#### 5.3 工程应用价值分析
内置的 5 大工程案例展示了从物理公式到器件的演进。例如智能窗户膜在可见与近红外波段的分段调控，能够让学生理解工程设计中的多波段多目标优化思路。

#### 5.4 与传统商业软件的关系
本项目开发的平台不是商业仿真软件（如有限元软件 COMSOL 等）的替代品，而是**作为教学演示与初步设计分析的轻量化补充方案**。对于平面多层膜及亚波长周期结构的零阶估算，本平台具有计算时间短、易部署的优势；但对于大倾角共振、任意二维/三维周期结构，仍需配合主流商业软件进行全波物理计算。

#### 5.5 平台局限性

##### 5.5.1 TMM 仅适用于平面层状结构
传输矩阵法假设介质界面为无限大且平整的平面，对于弯曲表面、粗糙边界、三维微纳柱或微球散射，TMM 在物理上不再适用。

##### 5.5.2 EMT 仅适用于亚波长近似
有效介质近似模块假设周期远小于波长（$P \ll \lambda$），仅计算零阶极化干涉。若周期参数与工作波长接近，由于发生波导共振、强多阶衍射激发，EMT 模型不再适用，需要采用更高阶的周期结构电磁求解方法。

##### 5.5.3 COMSOL CSV 不代表平台内部全波求解
平台外部数据读取管线仅用于导入外部仿真生成的 CSV 离散数据列并进行特征提取，平台内部计算引擎并不支持全波方程的数值离散物理求解。

#### 5.6 项目可行性分析
本平台纯 Python 科学计算框架开发，包体小、部署简便。通过单元测试套件保障计算逻辑，项目已预留 GitHub Actions 工作流用于后续持续集成验证；当前竞赛展示版主要以本地 pytest 单元测试和 smoke_test 基础自检结果作为可靠性验证依据，具备较高的开发稳定性和扩展可行性。

### 七、版本冻结与后续计划

#### 7.1 v0.2.1 竞赛展示功能冻结说明
为保证在竞赛演示和评审过程中的稳定性，本项目版本确立为 `v0.2.1`。
本版本正式进入功能冻结保护，以防止大范围逻辑重构带来不可预测的计算偏离或异常。

#### 7.2 冻结内容
冻结了新增的物理模块、新的数学物理计算公式、新的计算大类主线，以及除 TMM、EMT、COMSOL CSV 以外的额外电磁求解器开发。

#### 7.3 允许继续修改的内容
允许对现有文档、相对路径链接进行修正与拼写校正；允许对终端中文编码等 BUG 进行兼容性修复；允许增加新单元测试用例及补充冒烟测试数据；允许优化 Plotly 图表展示样式及课堂演示材料。

#### 7.4 不建议加入的新模块
1. **严格耦合波级次矩阵特征值求解器**：其傅里叶空间阶数多、矩阵复数本征解在极化方向易发散，容易引起计算不稳定。
2. **多层 Web 网页交互式控制台接口**：当前版本主要通过 Python 脚本、案例入口和 Plotly 交互图表完成仿真演示。后续可在不改变核心计算模块的基础上封装为轻量化演示界面。不建议在冻结期继续增加网页开发框架依赖包。
3. **厚度多参全局逆向设计反演算法**：多维反演具有不收敛与解的多值共存性，本版本已彻底移除相关逆向反演，禁止重新加载。

#### 7.5 后续发展方向
后续计划在核心 TMM 计算引擎保持稳定的前提下，可探索轻量化 Web 展示接口，以提高参数调节的可交互体验，并进一步研究将高阶傅里叶衍射解算器引入 1D 周期微纳结构的物理仿真中。
