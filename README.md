# PyThinFilm - 基于 Python 的薄膜光学仿真、设计与教学平台

> **2026 年全国大学生物理实验竞赛创新赛自选课题类创新作品 (v0.2.1 竞赛展示功能冻结版)**

本平台是一个轻量化、开源、可视化且计算逻辑可查的一维平面多层薄膜电磁计算及亚波长周期结构近似分析系统。平台自主实现了传输矩阵法（TMM）向量化求解器、真实材料色散插值库及有效介质理论（EMT）近似模块，不依赖 COMSOL 等大型商业软件的实时运行环境，为高校物理教学与光电工程设计提供轻量化验证方案。

## 一、项目定位与解决思路

### 1. 高校教学与实验中的现实困难
在《波动光学》及相关微纳光子学课程中，多层膜干涉的解析计算公式极其繁琐、概念抽象。例如，随着薄膜层数增加，菲涅尔反射系数在多个界面的相长/相消干涉叠加以及腔内电场空间分布变得难以直观推导。学生通常仅能接触到少层数理想常数介质的简化结论，对真实色散介质和微纳结构的电磁响应缺乏直观感知。此外，测量多层膜光谱所需的高真空镀膜机、椭偏仪等设备造价昂贵，难以大范围面向本科生开展自主实验，导致理论与工程实践脱节。

### 2. 开源平台的解决思路
PyThinFilm 平台通过将薄膜物理理论直接转化为透明的 Python 代码，实现了“正向仿真、交互设计、自检复验”一体化：
- **物理优先**：拒绝黑盒计算，所有核心计算（Fresnel、TMM、EMT）均为公式直接编码，逻辑透明可查。
- **降本增效**：基于 NumPy 向量化广播重构计算内核，计算耗时微秒级，低硬件门槛实现 Plotly 交互式课堂教学演示。
- **可信验证**：引入 270 个单元用例的 pytest 自动化测试套件与 `smoke_test` 自检机制，确保计算流程具有高物理一致性。

## 二、底层物理方程与理论基础

本平台物理仿真内核建立在严格的经典电磁场理论之上，各计算模块与物理方程映射关系如下：

![图 2-3：平台底层物理方程与模块实现映射图](docs/figures/fig_2_3_physics_mapping.png)

### 1. 麦克斯韦方程与切向边界条件
电磁波在平面多层介质中的传播由麦克斯韦旋度方程描述：
$$\nabla \times \mathbf{E} = -\mu \frac{\partial \mathbf{H}}{\partial t}, \quad \nabla \times \mathbf{H} = \epsilon \frac{\partial \mathbf{E}}{\partial t}$$

在两层平整介质界面处，电磁场的切向分量必须连续，这是特征矩阵递推的基础：
$$E_{1,t} = E_{2,t}, \quad H_{1,t} = H_{2,t}$$

### 2. 菲涅尔公式与相位延迟
根据边界连续性条件，对于 TE（S）偏振和 TM（P）偏振，单界面的振幅反射系数 $r$ 和透射系数 $t$ 分别满足：
$$r_s = \frac{n_1 \cos\theta_i - n_2 \cos\theta_t}{n_1 \cos\theta_i + n_2 \cos\theta_t}, \quad t_s = \frac{2n_1 \cos\theta_i}{n_1 \cos\theta_i + n_2 \cos\theta_t}$$
$$r_p = \frac{n_2 \cos\theta_i - n_1 \cos\theta_t}{n_2 \cos\theta_i + n_1 \cos\theta_t}, \quad t_p = \frac{2n_1 \cos\theta_i}{n_2 \cos\theta_i + n_1 \cos\theta_t}$$

平面简谐波在第 $j$ 层介质（折射率 $\tilde{n}_j(\lambda) = n_j(\lambda) - i\kappa_j(\lambda)$，物理厚度 $d_j$）中单次往返的传播相位延迟为：
$$\delta_j = \frac{2\pi}{\lambda} \tilde{n}_j d_j \cos\theta_j$$

### 3. 传输矩阵法（TMM）
在第 $j$ 层介质边界上，切向电磁场通过 $2 \times 2$ 特征矩阵 $M_j$ 关联：
$$\begin{pmatrix} E_{j-1} \\ H_{j-1} \end{pmatrix} = M_j \begin{pmatrix} E_j \\ H_j \end{pmatrix} = \begin{pmatrix} \cos\delta_j & \frac{i}{\eta_j}\sin\delta_j \\ i\eta_j\sin\delta_j & \cos\delta_j \end{pmatrix} \begin{pmatrix} E_j \\ H_j \end{pmatrix}$$
其中 $\eta_j$ 为斜入射光学导纳（TE 偏振为 $\tilde{n}_j\cos\theta_j$，TM 偏振为 $\tilde{n}_j/\cos\theta_j$）。对于 $N$ 层介质堆叠，其总特征矩阵为：
$$M = M_1 M_2 \cdots M_N = \begin{pmatrix} m_{11} & m_{12} \\ m_{21} & m_{22} \end{pmatrix}$$

振幅系数可通过入射光学导纳 $\eta_0$ 与基底光学导纳 $\eta_s$ 求解，最终得到反射率 $R$、透射率 $T$ 和吸收率 $A$：
$$R = |r|^2, \quad T = \frac{\text{Re}(\eta_s)}{\text{Re}(\eta_0)}|t|^2, \quad A = 1 - R - T$$

## 四、快速开始与使用指南

### 1. 环境准备
平台基于 Python 开发，环境要求为 Python 3.10+。请通过以下命令配置依赖环境：
```bash
pip install -r requirements.txt
```
若需要开启 Plotly 交互式可视化，请另外安装：
```bash
pip install plotly
```

### 2. 一键本地自检
在修改代码或开始演示前，建议通过以下脚本进行一键冒烟自检（Smoke Test），验证基础 API 的连通性及文件生成逻辑：
```bash
python smoke_test.py
```
全部校验通过后，终端将输出通过状态，自检日志将保存在 `docs/evidence/smoke_test_log.txt`。

### 3. 运行经典案例
- **运行基础教学案例（例如单层减反膜）**：
  ```bash
  python run_teaching_demo.py --case single_ar
  ```
- **运行真实材料数据库查询演示**：
  ```bash
  python run_material_library_demo.py
  ```
- **运行亚波长一维周期结构偏振极化演示**：
  ```bash
  python run_guided_grating_demo.py
  ```

默认输出目录为 `~/thinfilm_outputs`，也可以通过环境变量 `THINFILM_OUTPUT_DIR` 自定义指定。

## 五、经典案例与物理现象复验

本平台内置了丰富的仿真设计案例，涵盖经典的教科书物理模型、真实色散材料导入、五个工程设计场景，以及用于物理边界拓展的研究示例。

### 1. 基础物理现象复验
- **单层增透膜（反射相消干涉）**：在中心波长 $\lambda_0 = 550\text{ nm}$ 处满足 $nd = \lambda_0/4$ 和 $n = \sqrt{n_0 n_s}$ 振幅条件，复现反射率归零点。
- **多层高反膜（布拉格反射停带）**：交替堆叠 $N$ 周期 1/4 波长高低折射率膜层，演示电场相位累加相长干涉，展示反射停带（Bandgap）宽度随周期数指数级上升的演化趋势。
- **F-P 共振腔（窄带透射）**：构建 DBR-Cavity-DBR 谐振结构，当满足半波厚度相位匹配 $\Delta\phi = 2m\pi$ 时，在停带中央激发极窄（FWHM 仅几纳米）的共振透射峰。
- **斜入射偏振分裂（各向异性）**：展示 TE 与 TM 偏振光学导纳在斜入射角下的分裂现象，复现 TM 偏振特有的布儒斯特角反射归零。
- **腔内电场空间驻波**：支持递推计算多层介质中简谐场正反向叠加的驻波强局域化分布，直观演示缺陷态能量捕获。

### 2. 真实材料色散与吸收效应
实际物性表现出强烈的波长依赖极化（色散）和带间跃迁（吸收）。本平台集成了 RefractiveIndex.INFO 的实测 CSV 数据，利用 **Cubic Spline 三阶样条插值** 获得连续的 $\tilde{n}(\lambda) = n(\lambda) - i\kappa(\lambda)$ 参数：

![图 2-2：真实材料 nk 数据读取与插值流程图](docs/figures/fig_2_2_nk_flow.png)

通过 materials 模块，能够复现金属（Au, Ag）在紫外波段由消光系数 $\kappa$ 引起的反射衰减，以及半导体等介质的色散漂移，极大提升了仿真物理真实感。

### 3. 五个 TMM-only 工程设计案例
位于 `examples/applications/`，全部由独立 Python TMM 引擎驱动，生成高保真交互光谱 HTML：
1. **太阳能电池减反膜**：通过多层膜设计，在 AM1.5 太阳能吸收波段（300∼1100 nm）最小化积分反射，优化光电效率。
2. **三腔 WDM 窄带滤光片**：设计用于光通信波分复用（1540∼1560 nm）的多腔级联滤波，实现平顶陡峭带通。
3. **激光高反镜**：在 1064 nm 固体激光器波长下提供超窄带宽的高反射，抑制热负荷。
4. **相机镜头多层增透**：在可见光波段（400∼700 nm）实现全波段极低反射，消除重影鬼影，相比单层 AR 优势明显。
5. **智能节能窗膜**：设计在可见光波段高透、近红外波段（800 nm 以上）高反射的滤光膜，实现阻隔热辐射。

### 4. 物理边界拓展研究
- **日间被动辐射制冷 (PDRC)**：在 $\text{SiO}_2$ 与 $\text{TiO}_2$ 真实色散数据上进行结构设计，利用标准太阳光谱（ASTM G173 AM1.5 Global）和 $8\sim 13\ \mu\text{m}$ 大气红外窗口进行加权平均反射/发射积分，实现太阳波段极低积分吸收（$A < 5\%$）与红外窗口高发射（$\epsilon > 80\%$）。
- **Tamm 界面局域态探索**：计算布拉格高反镜与金属界面处的反射相位角角差，建立界面态存在性的 phase-wrap 筛选判据，演示 1D 局域边界模态。

## 七、平台物理与技术边界声明

为维护物理仿真的严谨性，PyThinFilm 严格声明当前版本（v0.2.1）的算法边界：

1. **核心计算逻辑**：传输矩阵法（TMM）是平台平面多层膜反射、透射、吸收和场强计算的唯一物理内核。
2. **一维周期结构极化分析**：`guided_grating/rcwa.py` 内部计算采用的是有效介质理论（EMT）零阶亚波长近似分析，将亚波长光栅极化为一个各向异性等效介质薄膜（TE 和 TM 表现为不同的折射率分量）。
   - **⚠ 特别注意**：当前代码模块不支持更高阶的 Fourier 空间衍射级次散射求解，**不对应严格的三维严格耦合波分析（RCWA）算法实现**。当周期周期 $P/\lambda \ge 0.5$ 导致衍射级次激发产生共振时，EMT 物理模型将失效。
3. **外部数据导入与分析**：平台内置的 `guided_grating/comsol_io.py` 用于读取和滤波外部 COMSOL 导出的 CSV 扫描光谱并进行特征提取（FWHM, Q, MAE 等）。**平台自身不含有任何全波三维 Maxwell 电磁有限元/差分数值求解器，不是 COMSOL 运行时替代品**。
4. **无虚构功能**：本仓库不涉及任何移动端 APP 源码、GUI 图形面板源码或持续集成通过的虚假截图。所有光谱特征曲线均基于底层 Python 代码计算数据导出并进行证据归档。
