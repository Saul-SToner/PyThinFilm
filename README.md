# PyThinFilm - 基于 Python 与传输矩阵法（TMM）的薄膜光学计算实验与教学平台

> **2026 年全国大学生物理实验竞赛创新赛自选课题类创新作品 (v0.2.1 功能冻结版)**

本平台是一个轻量化、开源、可视化且物理逻辑可查的一维平面多层薄膜电磁计算及亚波长周期结构等效分析系统。平台自主实现了平面多层膜的传输矩阵法（TMM）向量化计算模型、真实材料色散插值库及有效介质理论（EMT）近似模块。该系统脱离了对大型商业软件实时运行环境的依赖，专注于薄膜光学物理现象的计算复验，为后续轻量化展示界面预留了可复用的计算输出。

## 二、薄膜光学物理问题与 TMM 计算复验

本平台的核心计算逻辑基于薄膜电磁切向边界连续条件。当平面波入射至多层平面介质表面时，电磁场在每个平整界面发生无限次往复反射与折射，在界面外侧干涉叠加。

系统采用一维特征矩阵级联算法（TMM）求解该物理过程，其计算管线如下：

![图 2-1：传输矩阵法 TMM 计算流程示意图](docs/figures/fig_2_1_tmm_flow.png)

对于第 $j$ 层平面介质薄膜，其左边界处的切向场（电场 $E_{j-1}$，磁场 $H_{j-1}$）与右边界处的切向场（$E_j$，$H_j$）通过一维特征矩阵 $M_j$ 关联：
$$
\begin{pmatrix} E_{j-1} \\\\ H_{j-1} \end{pmatrix} = M_j \begin{pmatrix} E_j \\\\ H_j \end{pmatrix} = \begin{pmatrix} \cos\delta_j & \frac{i}{\eta_j}\sin\delta_j \\\\ i\eta_j\sin\delta_j & \cos\delta_j \end{pmatrix} \begin{pmatrix} E_j \\\\ H_j \end{pmatrix}
$$

其中 $\delta_j = \frac{2\pi}{\lambda} \tilde{n}_j d_j \cos\theta_j$ 为平面简谐波的传播相位延迟，$\eta_j$ 为第 $j$ 层介质的斜入射光学导纳（对于 TE 偏振为 $\tilde{n}_j\cos\theta_j$，TM 偏振为 $\tilde{n}_j/\cos\theta_j$）。级联所有 $N$ 层介质得到系统总传输矩阵：
$$
M = M_1 M_2 \cdots M_N = \begin{pmatrix} m_{11} & m_{12} \\\\ m_{21} & m_{22} \end{pmatrix}
$$

由入射光学导纳 $\eta_0$ 与基底导纳 $\eta_s$ 计算得到振幅系数 $r$ 和 $t$。最终解得光谱反射率 $R$、透射率 $T$ 和吸收率 $A$：
$$
R = |r|^2, \quad T = \frac{\text{Re}(\eta_s)}{\text{Re}(\eta_0)}|t|^2, \quad A = 1 - R - T
$$

系统利用 NumPy 向量化广播重构计算内核，消除了传统的波长和角度循环，在典型教学案例中具有较快的参数扫描速度。

## 三、底层物理方程与模块映射体系

为了保证仿真算法的可信度，平台明确梳理了底层电磁场物理方程在计算代码中的映射实现关系：

![图 2-3：平台底层物理方程与模块实现映射图](docs/figures/fig_2_3_physics_mapping.png)

1. **麦克斯韦旋度方程**：麦克斯韦方程描述了一维简谐平面波在无源介质中的基本电磁物理规律，是波动光学推导的源头。
2. **切向边界条件**：界面处切向电场和磁场连续（$E_t$ 连续，$H_t$ 连续），用于构建单层及多层级联的递推边界条件。
3. **能量守恒约束**：定义 $R+T+A \equiv 1.0$ 的强物理一致性边界，作为自动化测试的核心断言，防止数值溢出。
4. **一维腔内局域场**：基于边界条件进行正向和反向场强递推，求解一维腔内不同深度 $z$ 处驻波的干涉分布 $E(z)$。

## 四、真实材料 nk 极化色散与吸收

实际光学薄膜材料具有随波长变化的物理极化（色散）以及带间跃迁（吸收损失）。理想的折射率常数近似在短波区或强吸收区会产生显著的仿真误差。

平台集成了实测物理常数数据库，其数据检索与插值处理逻辑如下：

![图 2-2：真实材料 nk 数据读取与插值流程图](docs/figures/fig_2_2_nk_flow.png)

- **线性插值**：将离散实测 n/k 数据映射到目标仿真波长网格；默认禁止超出源数据有效范围外推；结合缓存机制避免重复读盘。
- **复折射率符号口径**：由 n(λ) 与 κ(λ) 构造复折射率 $\tilde{n}(\lambda) = n(\lambda) + ik(\lambda)$ 并输入 TMM，其虚部用于描述材料吸收；具体正负号取决于时间因子约定。
- **消光系数吸收模型**：消光系数 $\kappa(\lambda) > 0$ 决定了传播相位延迟 $\delta$ 中的虚部，实现了物理上带间吸收和能量衰减的复现。
- **LRU 缓存控制**：使用单次读盘与内存字典缓存，避免由于重复进行磁盘读取导致的性能瓶颈。

## 五、亚波长零阶 EMT 近似与外部 CSV 导入

一维周期性微纳周期结构通常需要通过繁琐的网格离散和矩阵特征值求解。本平台引入了针对亚波长结构的近似模块与外部仿真比对管线，实现轻量化趋势分析：

### 1. 有效介质理论（EMT）近似
对于一维周期结构光栅，当其周期 $P$ 远小于入射电磁波长 $\lambda$ （静态区，$P \ll \lambda$）时，光栅层可等效为一个各向异性的均质单层薄膜。
在零阶有效介质理论下，对于 TE 和 TM 偏振，光栅层可等效为以下折射率介质：
$$
n_{eff,TE}^2 = f n_{high}^2 + (1 - f) n_{low}^2, \quad \frac{1}{n_{eff,TM}^2} = \frac{f}{n_{high}^2} + \frac{1 - f}{n_{low}^2}
$$
其中 $f$ 为高折射率介质所占的体积占空比。本平台基于此等效值，使用传输矩阵法计算单层各向异性薄膜，以演示亚波长光栅的偏振极化与双折射干涉特性。

### 2. 外部 COMSOL 数据导入与特征分析
为了展示超出 EMT 近似适用范围（如周期与波长可比拟并激发波导共振）的物理特性，平台设计了外部 CSV 数据解析读取管线。
通过 Pandas 解析包含复杂表头的外部仿真 CSV，算法可自动提取曲线中的半高全宽（FWHM）、计算品质因子 $Q = \lambda_0/\text{FWHM}$ 以及测量数据与理论模型之间的平均绝对误差（MAE），以此作为参数筛选指标。

## 六、计算实验案例与应用展示

基于平台 TMM 计算内核，平台内置了从经典教学演示到工程案例的计算实验设计。

### 1. 经典薄膜现象复验
- **单层增透膜**：在设计中心波长处复现相消干涉导致的反射率极小值。
- **布拉格反射镜（DBR）**：展示随着交替膜层周期数增加，光子带隙（停带）反射率呈指数级上升的相长干涉物理图景。
- **F-P 微腔透射**：复现双 DBR 腔结构中由半波相位匹配激发的超窄带高透射共振峰。
- **斜入射偏振各向异性**：斜入射角下 TE/TM 极化导纳分裂，演示 TM 波特有的布儒斯特角反射归零。
- **微腔腔内电场驻波**：直观求解并展示谐振波长下腔内空间驻波的极强局域化分布。

### 2. 5 个 TMM-only 工程应用案例
位于 `examples/applications/`，专注于物理指标分析与光谱图表导出：
- **太阳能电池减反膜**：在 300–1100 nm 太阳光波段最小化积分反射，优化电池光电收集效率。
- **波分复用（WDM）三腔滤波器**：在光通信频段（1540–1560 nm）实现平顶陡峭带通。
- **1064 nm 固体激光器反射镜**：为特定单色波长提供超窄带宽的高反射，抑制腔镜热负载。
- **相机镜头多层增透膜**：对比多层与单层 AR 在可见光波段（400–700 nm）的反射谱差异。
- **智能窗户节能膜**：可见光波段高透、近红外波段（800 nm 以上）高反射，复现日光阻隔效果。

### 3. 前沿拓展研究案例
- **被动辐射制冷 (PDRC)**：在 $\text{SiO}_2$ 与 $\text{TiO}_2$ 真实色散数据上进行结构设计，利用标准太阳光谱（ASTM G173 AM1.5 Global）和 8–13 μm 大气窗口加权平均，计算日间被动制冷评分。
- **Tamm 界面局域态探索**：计算布拉格反射镜与吸光层界面反射相位差，展示 1D 边界模式的极值相位匹配条件。

## 七、自动化测试与自检证据链

为防范计算过程中的数值溢出、理论失效或插值震荡，平台设计了闭环的自检与测试证据链：

![图 3-5：测试与证据材料生成流程图](docs/figures/fig_3_5_test_flow.png)

1. **一键本地自检（Smoke Test）**：
   一键运行以验证基础接口的连通性、各案例入口参数验证及结果文件的正确生成：
   ```bash
   python smoke_test.py
   ```
2. **pytest 单元测试套件**：
   包含 11 个大类测试文件共计 **270 个单元用例**，自动校验能量守恒性（$R+T+A \equiv 1.0$）、单层退化极值及偏振一致性：
   ```bash
   python -m pytest tests/ -v
   ```

自检输出日志与测试结果汇总将统一归档至 `docs/evidence/` 目录。

## 八、项目物理与技术边界声明

本平台作为学术复验与辅助教学工具，严格厘清技术与物理边界：

1. **核心计算局限性**：传输矩阵法（TMM）是一维层状平面结构计算的唯一物理内核，不支持任何非平整界面、粗糙表面散射或三维空间离散计算。
2. **EMT 模块边界**：`guided_grating/rcwa.py` 内部计算仅在零阶有效介质理论近似下提取各向异性等效介质折射率并输入 TMM 计算。该文件名沿用了早期开发阶段的模块命名；**当前版本平台不对应严格的三维严格耦合波分析（RCWA）算法实现**。当周期 $P/\lambda \ge 0.5$ 激发多阶衍射时，该 EMT 模型将完全失效。
3. **外部仿真 CSV 定位**：COMSOL CSV 数据解析读取器仅用于导入前期复杂结构探索导出的 CSV 数据并计算 FWHM、Q 值、MAE 误差。**本平台不包含任何三维 Maxwell 有限元/差分数值求解器，不具备商业电磁软件替代关系**。
4. **无虚构功能与状态**：本仓库中不包含任何移动端 App 界面、GUI 渲染源码，亦不包含任何 GitHub Actions 持续集成已通过的虚假徽章或状态声明。所有物理谱线均由本地 Python 计算脚本生成并进行真实证据归档。
